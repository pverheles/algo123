Definition

Dynamic Programming (DP) solves problems with overlapping subproblems and optimal substructure. It stores results of subproblems to avoid recomputation.

How to detect DP

  • Subproblems repeat during a naive recursion (overlap).
  • Optimal solution can be expressed via optimal solutions to subproblems (substructure).
  • State can be defined with a few parameters; transitions combine smaller states.
